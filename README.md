# Web QQ 模拟

<label>build status: <label style=" background-color:#48C21A;border-radius:3 3">MyBatis整合结束</label></label>

### 运行环境
- firefox
- JDK 1.8
- Maven
- MyEclipse

### 项目介绍
- 项目模拟WebQQ
- 前端
	- twemoji 提供表情功能
- 后台
	- WebSocket提供客户端之间的通信
	- MySQL为数据库，MyBatis提供与数据库通信
	- Spring MVC

### 功能
1. emoji表情收发
2. 历史聊天记录查看
3. 更换头像(头像由系统提供)
4. 添加/删除好友
